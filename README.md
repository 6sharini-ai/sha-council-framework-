# 🏛️ Sha's Council Framework

> Open framework for collaborative AI reasoning — where diverse AI agents debate, synthesize, and help humans make wiser decisions.

## 🎯 The Problem
Single-perspective AI assistants optimize for coherence, not wisdom. Real decisions benefit from structured tension: challenge + support + innovation + synthesis.

## ✨ The Solution
A lightweight protocol for multi-agent debate:
1. **Frame** the question (neutral context)
2. **Diverge** with specialized perspectives (Challenger, Supporter, Innovator...)
3. **Converge** with explicit tradeoffs (Synthesizer)
4. **Decide** with human agency (You)

## 🚀 Quick Start (Gemini)
1. Create a Gem named "Sha's Council"
2. Paste [instructions/v3.1.md](instructions/v3.1.md) into Instructions field
3. Type `/council quick: [your decision]`
